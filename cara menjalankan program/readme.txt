cara menjalankan PEDeS

setelah mengimport database,

1. buat folder baru di htdocs dengan nama rpl
2. pindahkan isi folder 'index v2' ke dalam folder rpl
3. buat folder pengingat didalam folder rpl, lalu pindahkan isi folder 'pengingat v2' kedalam folder pengingat
4. buat folder login didalam folder rpl, lalu pindahkan isi folder 'login dan register v2.0' kedalam folder login
5.buat folder saran didalam folder rpl, lalu pindahkan isi folder 'saran v2.0' kedalam folder saran
6. buat folder tanggapan didalam folder rpl, lalu pindahkan isi folder 'tanggapan02' kedalam folder tanggapan
7. buat folder history didalam folder rpl, lalu pindahkan isi folder 'history02' kedalam folder history

NOTE:di menu login, jika ingin mengakses page admin, gunakan nik=123, password=anu. akun admin tidak bisa dibuat melalui form register, jika membuat akun admin harus dibuat melalui database. 
form register hanya bisa membuat akun user.
